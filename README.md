Power BI Graduate Admissions Dashboard
Interactive multi-page Power BI dashboard analyzing University of Louisville graduate admissions data, built as a capstone project for MSBA 610 (Data Visualization with Power BI).
Overview
This dashboard helps university leadership analyze admissions trends, applicant demographics, and recruitment effectiveness across programs. It was designed for executive-level review with an emphasis on clarity, interactivity, and actionable KPIs.
What I Built

Performed Power Query ETL to clean and transform raw admissions data
Created DAX measures for conversion rates, YTD trends, and demographic breakdowns
Designed interactive slicers, map charts, matrix tables, and KPI cards across multiple report pages
Applied consistent UofL branding (colors, fonts, themes) throughout

Tools

Power BI Desktop (Power Query, DAX)
Microsoft Excel (source data)

Key Takeaways

Iteratively refined slicer placement to reduce clutter while preserving interactivity
Identified opportunities for future enhancements including predictive modeling and marketing analytics integration
